Vistumbler v10.8.2

Released: 05/03/2023

Vistumbler is an open-source wireless network scanner for Windows. It maps and visualises nearby access points using wireless and GPS data.

EXE Installer ZIP Source ZIP Portable

About

Vistumbler is written in AutoIt for Windows. The main purpose of Vistumbler is to map and visualise access points around you based on wireless and GPS data collected.

Features

  • Windows 11 and Windows 10. .NET Framework 4 required.
  • Uses the Windows Native WiFi API or netsh to find access points.
  • GPS support (COM/NMEA receivers).
  • Export/import of APs and KML/GPX export.
  • Live Google Earth tracking and audio alerts.
  • Open Source (GPLv2). Source: GitHub.

Notes

  • For best results, disconnect from APs while scanning.
  • Change language via Settings or help create a language pack on the forum.
  • Ensure the correct network adapter is selected if no APs show.
