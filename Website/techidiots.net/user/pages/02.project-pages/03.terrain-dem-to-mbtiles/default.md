**Project**: JAXA AW3D30 to MBTiles  
**Description**: A collection of scripts to convert JAXA AW3D30 geotiff files to terrain raster-dems, color-releif, and contour lines.  
**Link**: [https://github.com/acalcutt/jaxa_AW3D30_to_MBTiles](https://github.com/acalcutt/jaxa_AW3D30_to_MBTiles)  

**Project**: GEBCO_to_MBTiles  
**Description**: A collection of scripts to convert GEBCO bathymetry files to terrain raster-dems and color relief.  
**Link**: [https://github.com/acalcutt/GEBCO_to_MBTiles](https://github.com/acalcutt/GEBCO_to_MBTiles)  

**Project**: rio-rgbify  
**Description**: A custom version of mapbox rio rgbify with added fixes and deduplication. there is also dataset merging in the 'merge'; branch being tested.  
**Link**: [https://github.com/acalcutt/rio-rgbify](https://github.com/acalcutt/rio-rgbify) ( [test version with added merging](https://github.com/acalcutt/rio-rgbify/tree/merge) )
