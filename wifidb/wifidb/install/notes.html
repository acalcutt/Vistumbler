WifiDB [0.16 Build 3]
Installation / Upgrade / Patch Notes

    Install
  1. The install script Here will install WiFiDB 0.16 Build 1 on your server.
  2. It is NOT recommended to run an install over the internet, it should only be run locally, after the install is successful delete the Install folder.
  3. You will need Root access or at least GRANT permissions on your MySQL server to install the WiFiDB.
  4. You will need GD2 to beable to use the Signal Graphing feature, if you do not have this you must install/enable it.
  5. From my development experiments (and wanting to know why it was taking so long to drop a db with 30k+ tables) I have seen MySQL (and PHP) run much much much faster on Linux then on Windows, for some time I was running my development and 'production' enviroments as windows only, I setup a very Modest system (PIII 450mhz w/ 768Mb of RAM, running FC9) and it was light speeds faster then the windows box (same hardware just server 2003).
    Upgrade
  1. The Upgrade script Here will upgrade the config file, and some tables in the database.
  2. Upgrade to [0.16 Alpha Build 3].
  3. As always, backup your data, before doing any work on it.
