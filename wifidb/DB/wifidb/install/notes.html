WifiDB Installation Notes

  1. It is NOT recommended to run an install over the internet, it should only be run locally, after the install is successful delete the Install folder
  2. You will need Root access or at least GRANT permissions on your MySQL server to install the WiFiDB
  3. You will need GD2 to beable to use the Signal Graphing feature, if you do not have thism you must install/enable it.
  4. From my development experiments (and wanting to know why it was taking so long to drop a db with 30k+ tables) I have seen MySQL (and PHP) run much much much faster on Linux then on Windows, for some time I was running my development and 'production' enviroments as windows only, I setup a very Modest system (PIII 450mhz w/ 768Mb of RAM, running FC9) and it was light speeds faster then the windows box (same hardware just server 2003)
